1. 로그인 및 회원가입

1) 로그인

2) 회원가입

  • 사이트 접속

    https://histutor.lifove.net/로 접속해서 한동 이메일을 이용해서 로그인 합니다.

다른 이메일로 로그인을 시도할 경우 로그인이 정상적으로 이루어지지 않으니 꼭 한동 이메일로 로그인합니다.

  • 회원가입

첫 로그인 시, 회원가입 절차로 바로 넘어갑니다.

  • 학번 + 이름 확인 및 수정

자신의 학번과 이름이 맞는지 확인하시고,

틀렸을 경우 학번+이름 형식으로 수정해주세요.

예) 17학번 김종은 → 17김종은

  • 핸드폰 번호 입력

아래 칸에 핸드폰 번호를 입력하고 제출 버튼을 눌러서 회원가입 절차를 마칩니다.

2. 튜터링 참여

튜터가 생성한 튜터 세션이나 코딩 아워 튜터링에 참여하실 수 있습니다.

1) 튜터링 리스트 확인

메인 화면에서 진행되고 있는 튜터링들 중에서 원하는 튜터링의 제목을 클릭합니다.

2) 대기방에서 대기하기

Histutor의 튜터링은 1대1 방식으로 이루어집니다.

튜터가 자신을 부를 때까지 대기방에서 기다립니다.

여기서 총 대기자 수, 내 대기 순서 및 내 앞과 뒤에 몇 명의 튜티들이 대기하고 있는지 확인할 수 있습니다.

자신의 차례가 되면 이동한다는 알림과 함께 화면이 튜터링방으로 자동으로 이동됩니다.

❗ 주의할 점❗ 대기방을 나가거나 새로고침 할 경우 순서가 밀려나게 되니 이 점 유의해 주세요 !

3) 튜터링 참여

이제 튜터와 튜터링을 시작하게 되었습니다! 🥳

도움이 필요한 부분이나 궁금한 점을 채팅을 통해 물어보세요 🙌

질문을 다 마치고 자신의 차례가 끝날경우 튜터에 의해서 자동으로 메인화면으로 나가게 됩니다.

3. 튜터링 모집

1) 튜터링 방 생성

도움이 필요한 부분에 대해서 튜터들에게 도움을 요청할 수 있는 방을 만듭니다.

  • 튜터링 방 만들기

    메인 페이지 상단의 새로운방 만들기를

    클릭합니다.

  • 제목 입력

    어느 부분에 도움이 필요한지 알 수 있도록 최대한 자세히 제목을 입력합니다.

    예) MySQL 설치 이후 root 계정에 접근이 안되는데 도와주세요.

  • 튜터링 방 생성 완료

이제 튜터링 방이 만들어졌습니다!

방이 생성되고 방의 상태는 튜터를 모집중으로 설정됩니다.

2) 튜터링 시작

도움이 필요한 부분에 대해서 들어온 튜터와 이야기를 나눈 뒤 도움을 받으시면 됩니다.

튜터가 해당 문제에 대해서 도움을 줄 수 있다고 판단하면 해당 채팅방의 튜터로 자신을 지정하게 되고 방의 상태가 튜터링 진행중으로 바뀌게 됩니다.

문제가 해결되고 나면 튜터가 튜터링을 종료합니다.

3) 보고서 작성 → 선택 사항

튜터링이 어땠는지 후기를 남겨주세요.

내용은 자유롭게 작성하셔도 됩니다.

간단한 가이드 라인으로는 도움 받았던 내용이나 튜터가 어떻게 도와줬는지, 도움이 되는지 등에 대한 내용이 있습니다.

4. 마이페이지

마이페이지에서는 지금까지 내가 모집한 튜터링들 또는 내가 참여한 튜터 세션의 목록을 확인할 수 있습니다.

5. 대댓글 기능 🤩

Histutor의 대댓글 기능을 소개합니다.

1) 채팅을 더블클릭으로 선택

답을 남기고 싶은 채팅에 더블 클릭✨을 하면, 대댓글 창이 열리게 됩니다.

2) 댓글에 대한 답을 작성

남기고 싶은 말을 작성하시면, 이쁜 대댓글 창을 확인하실 수 있습니다.
