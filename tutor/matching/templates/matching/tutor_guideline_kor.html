1. 로그인 및 회원가입

1) 로그인

https://histutor.lifove.net/에 접속해서 한동 이메일로 로그인 합니다.

2) 회원가입

첫 로그인 시 회원가입 절차가 이루어집니다.

  • 학번 + 이름 확인 및 수정

자신의 학번과 이름이 맞는지 확인하시고,

틀렸을 경우 학번+이름 형식으로 수정해주세요.

예) 17학번 김종은 → 17김종은

  • 핸드폰 번호 입력

핸드폰 번호를 입력하고, 회원가입을 완료합니다.

2. 튜터 신청

  • 마이페이지 이동

    상단 오른쪽의 MY PAGE 버튼을 눌러 마이페이지로 이동합니다.

  • 튜터 신청하기

    마이페이지에서 튜터 신청하기 버튼을 클릭합니다.

    신청 이유를 간단하게 작성합니다.

  • 관리자가 신청을 승인하면, 튜터 권한을 얻게 됩니다.

3. 튜터링 모집

튜터 세션이나, 코딩 아워 튜터링을 모집합니다.

1) 튜터링 방 생성

  • 상단의 새로운 방 만들기를 클릭합니다.
  • TA세션 버튼을 클릭합니다.

  • 튜터링 정보를 입력합니다.

  • 방 이름

    어떠한 튜터링인지를 자세하게 입력합니다.

  • 세션 유형

    대면, 비대면, 병행 중에 선택하세요.

    대면 → [오프라인]으로 진행

    비대면 → [온라인]으로 진행

    병행 → [온라인], [오프라인] 진행

  • 오프라인 장소

    오프라인으로 튜터링을 진행한다면,

    장소를 입력합니다.

  • 시작 시간, 종료 시간

    튜터링의 시작 시간과 종료 시간을

    입력합니다.

  • 튜터링 방 생성 완료

    모든 준비를 마쳤습니다 👏

    이제 본격적으로 튜터링을 시작해봅시다 !

2) 새 튜터링 시작

Histutor의 튜터링은 1대1 방식으로 이루어집니다.

대기 중인 튜티 수 에서 현재 이 튜터링에 참여하고자 대기하는 튜티가 몇 명인지 알 수 있습니다.

  • 새 튜터링 시작 버튼을 눌러, 대기중인 튜티를 불러오세요!

  • 새 튜터링 시작 버튼을 다시 누르면...

    → 이전 튜터링이 종료되고

    → 새로운 튜티와의 튜러링이 시작됩니다.

3) 튜터링 종료

모든 튜터링이 끝나면, 튜터 세션 종료 버튼을 눌러 튜터 세션을 종료합니다.

4) 보고서 작성

튜터링에 대한 보고서를 작성하실 수 있습니다.

→ 간단하게 어떤 내용의 튜터링이었는지 적어주세요!

5) 보고서 확인

  • 내가 작성한 보고서의 리스트는

    튜터링 방 내부에 보고서 보기 버튼을 누르면 확인할 수 있습니다.

  • 보고서는 관리자와 자신만 확인할 수 있습니다.

4. 오프라인 튜터링 등록

오프라인에서 튜터링을 진행하셨나요?

그렇다면, 바로 보고서를 작성하시면 됩니다!

튜터링 방 생성 → 튜터링 종료 → 보고서 작성 하여 진행한 튜터링을 남겨주세요 🙌

➡️

5. 튜터링 참여

튜티가 도움을 요청하는 튜터링에 참여하실 수 있습니다.

1) 튜터링 리스트 확인

홈 화면에서 튜티가 도움을 요청하는 튜터링을 확인하실 수 있습니다.

모집중인 튜터링이 있는지 확인합니다.

2) 튜터링 시작

튜티와 인사를 나누고, 우측의 튜터링 시작 버튼을 눌러 튜터링을 시작해보세요!

3) 튜터링 취소 or 튜터링 완료

어떠한 이유로 튜터링을 취소한다면, 튜터링 취소 버튼을 눌러주세요.

튜티와의 튜터링이 종료되었다면, 튜터링 완료 버튼을 눌러주세요.

4) 보고서 작성

튜터링이 완료되었으면, 튜터링에 대한 보고서를 작성하실 수 있습니다.

→ 간단하게 어떤 내용의 튜터링이었는지

→ 몇 분간 진행하였는지 적어주세요!

5) 보고서 확인

  • 내가 작성한 보고서의 리스트는

    튜터링 방 내부에 보고서 보기 버튼을 누르면 확인할 수 있습니다.

  • 보고서는 관리자와 자신만 확인할 수 있습니다.

6. 마이페이지

마이페이지에서 그동안의 내 이력을 확인할 수 있습니다.

적극적으로 참여한 자신에게 뿌듯함을 느껴보세요 ! 🤗

7. 대댓글 기능 🤩

Histutor의 대댓글 기능을 소개합니다.

1) 채팅을 더블클릭으로 선택

답을 남기고 싶은 채팅에 더블 클릭✨을 하면, 대댓글 창이 열리게 됩니다.

2) 댓글에 대한 답을 작성

남기고 싶은 말을 남겼다면, 이쁜 대댓글을 확인하실 수 있습니다.
